# BF OCAML
This was my first ocaml project. I am learning ocaml because I thought it would be fun, it is really fun.
It works perfectly fine, however it only interprets the file 'test.bf'
It has all of the bf features `+-<>.,[]` I think the way I did it is probably not the way most people would have but I like my method.
